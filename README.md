# SPCC_basic_codes
Contains Codes for Semester VI:-System Programming and Compiler Construction
The following codes are present:-
>lexical.java:-Lexical analyzer:Generating a stream of tokens from an i/p program.
>FirstFollow.java:Finding the first and follow of non terminals
>codegeneration.java:Generating code from given statements
>leftrecursion.java:removing direct and indirect left recursion
>macro.java:Macro processor program
>operator_precedence.java:OP parser for the grammars
>recursivedescentparser.java:Implementation of a recursive descent parser
>twopassassembler.java:Implementation of two pass assmembler
